## 1.16.0 (April 23, 2026)

FEATURES:

* querycheck: Add new `ExpectTotalLengthForMatching` querycheck for validating the number of found resources for multiple list blocks ([#607](https://github.com/hashicorp/terraform-plugin-testing/issues/607))
* helper/resource: Added new `GenerateConfig` testing mode to `TestStep` which can be used to test the generated resource configuration during `ImportState` testing mode. ([#638](https://github.com/hashicorp/terraform-plugin-testing/issues/638))

ENHANCEMENTS:

* helper/resource: Added `DefaultWorkspaceOnly` to enable testing state store implementations that don't support multiple workspaces. This is achieved by only running Terraform commands with the `default` workspace. ([#633](https://github.com/hashicorp/terraform-plugin-testing/issues/633))

BUG FIXES:

* querycheck: Fix behaviour of `ExpectLength` querycheck to match on the provided resource address before comparing the count ([#604](https://github.com/hashicorp/terraform-plugin-testing/issues/604))
* querycheck: Fix behaviour of `ExpectLengthAtLeast` querycheck to match on the provided resource address before comparing the count ([#607](https://github.com/hashicorp/terraform-plugin-testing/issues/607))
* dep: updated `hc-install` dependency to `0.9.4` to prevent openpgp key expiration errors. ([#647](https://github.com/hashicorp/terraform-plugin-testing/issues/647))
